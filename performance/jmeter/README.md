## Acerca del archivo docker

# para la ejecución:

- para ejecutar el archivo docker, dar permisos de administrador. Poner en la linea de 
comandos: chmod +x run.sh

- para ejecutar el archivo:
./run.sh

# los resultados son guardados en la carpeta results-docker

- para poder correr jmeter se uso su respectiva imagen de docker, sostenida por la 
comunidad de docker hub. 

# para ver el contenido de run.sh
- correr el suguiente comando en el directorio en el cual esta el ejecutable:
cat run.sh
